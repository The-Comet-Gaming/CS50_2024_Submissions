Overview

This website is for Problem Set 8 of CS50's Introduction to Computer Science where this problem set's focus is on using the main components of frontend web development those components being HTML, CSS, and JAVASCRIPT. The best example I have heard of the relationship between the 3 components is when you compare them to a human body. The HTML is the skeleton of the body as it largely determines the structure of the entire page, the JAVASCRIPT is the internal organs and musclature because it gives the skeleton (HTML) functionality that it wouldn't have on it's own, and finally the CSS is the skin, hair, eyes, etc which gives the latter two an appealing visual appearance that can be tweaked to fit a certain visual asthetic. For example I chose the colour Teal as a main colour for these webpages as it's a favourite colour of mine.
